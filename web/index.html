Servidor Web del Grupo 4

Ejercicio 5: Servidor Nginx Containerizado

Este servidor web forma parte del proyecto académico donde se implementa un servicio web usando Docker y Nginx. El objetivo es comprender el despliegue de aplicaciones en contenedores y la configuración de un servidor web minimalista.

Tecnologías utilizadas

  • Nginx: Servidor web utilizado para servir la página.
  • Docker: Contenerización del servidor y despliegue portátil.
  • Linux: Sistema operativo base del entorno.
  • HTML5: Estructura de la página web.

Descripción del Proyecto

El contenedor está configurado para servir este archivo index.html como contenido principal. Además, se trabajó en la creación del Dockerfile, construcción de la imagen y ejecución del contenedor mapeando el puerto del servidor Nginx.

Este entorno demuestra el flujo básico de creación y despliegue de servicios web usando tecnologías modernas.
